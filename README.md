## Installation

Make sure you have NodeJS installed (version 5+).
```shell
npm install
```

## Build
For minified/uglified/bundled/production-like build run:
```shell
npm run build
```
or
```shell
gulp build --prod
```

For developement/debugging run:
```shell
gulp build
```

## Watch
```shell
npm run watch
```
or
```shell
gulp watch
```
